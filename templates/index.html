Predicción Temprana de Enfermedad Cardíaca

Este proyecto tiene como objetivo desarrollar un modelo basado en minería de datos capaz de predecir la presencia de enfermedades cardíacas en pacientes utilizando información clínica.

Problemática

Las enfermedades cardíacas representan una de las principales causas de mortalidad a nivel mundial. En muchos casos, la detección se realiza cuando la enfermedad ya se encuentra en un estado avanzado, lo que reduce las posibilidades de tratamiento efectivo.

Por ello, es necesario aplicar técnicas de minería de datos que permitan analizar variables clínicas y descubrir patrones que indiquen la probabilidad de que un paciente desarrolle una enfermedad cardíaca.

Objetivo del proyecto

Desarrollar un modelo de minería de datos que permita predecir la presencia de enfermedad cardíaca a partir de variables como edad, colesterol, presión arterial, frecuencia cardíaca máxima y otros indicadores clínicos relevantes.

Minería de Datos

La minería de datos se enfoca en el descubrimiento de patrones y relaciones dentro de los datos. En este proyecto se utiliza para construir modelos predictivos que ayuden a clasificar pacientes en dos grupos: con o sin enfermedad cardíaca.
