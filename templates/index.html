Análisis de Enfermedad Cardíaca mediante Minería de Datos

Este proyecto tiene como objetivo aplicar técnicas de minería de datos para analizar información clínica de pacientes e identificar patrones asociados a la presencia de enfermedades cardíacas.

Problemática

Las enfermedades cardíacas representan una de las principales causas de mortalidad a nivel mundial. En muchos casos, la detección se realiza cuando la enfermedad ya se encuentra en un estado avanzado, lo que reduce las posibilidades de tratamiento efectivo.

Por ello, es necesario aplicar técnicas de minería de datos que permitan analizar variables clínicas y descubrir patrones que indiquen el nivel de riesgo de un paciente.

Objetivo del proyecto

Analizar variables clínicas como edad, colesterol, presión arterial y frecuencia cardíaca para identificar patrones y clasificar a los pacientes según la posible presencia de enfermedad cardíaca.

Minería de Datos

La minería de datos se enfoca en el descubrimiento de patrones, relaciones y tendencias dentro de los datos. En este proyecto se utilizarán técnicas de clasificación y análisis de datos para apoyar la toma de decisiones en el ámbito médico.
